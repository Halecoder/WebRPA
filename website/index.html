核心特性

为效率而生的每一处细节

从可视化编排到 AI 辅助，WebRPA 把复杂的自动化做成了拖拽这么简单。

零代码可视化

拖拽模块、连线成流，所见即所得。不写一行代码，也能搭出复杂自动化逻辑。

AI 小助手 · 自愈

对话式理解需求，自动搭建工作流；运行失败时自动诊断 → 修复 → 重跑，最多三轮。

544 个模块

网页、桌面、手机、Excel、数据库、AI、媒体、通知……覆盖 95% 自动化场景。

开箱即用

内置 Python 3.13 与 Node.js 运行环境，解压双击启动器即可运行，无需任何额外配置。

完全离线

所有资源本地化，无需外网连接，天然支持内网与局域网部署，数据不出本机。

智能搜索

模块与文档支持中文、拼音、拼音首字母模糊搜索，几个字母就能定位想要的能力。

变量自动补全

模块创建即内置默认变量，智能补全列表实时识别，引用变量再也不用记名字。

MCP 扩展

支持 Model Context Protocol 服务器接入，模块化架构可自定义开发，能力无限延伸。

可视化流程图

内置 Mermaid 流程图，一键生成直观的工作流逻辑视图，复盘与分享都更轻松。

常见问题

你可能想问的

WebRPA 需要编程基础吗？
完全不需要。通过拖拽模块、连线配置即可构建工作流；如果遇到复杂需求，还可以让内置 AI 小助手帮你自动搭建。
支持哪些操作系统？
目前仅支持 Windows 10 / 11。项目自带 Python 3.13 与 Node.js 运行环境，无需额外安装。
需要联网才能使用吗？
不需要。所有资源均本地化运行，可完全离线使用，天然支持内网与局域网部署。仅在调用外部 AI 服务时才需要对应的网络。
AI 小助手收费吗？
小助手功能本身免费。它采用 OpenAI 兼容协议，你只需填入自己的 API（OpenAI / 智谱 / Deepseek / Ollama 等），也可使用本地模型完全离线。
可以用于商业项目吗？
个人与非商业用途完全免费。商业场景请遵循 AGPL-3.0 协议，或联系作者获取商业授权以免除开源义务。

准备好让重复工作自己跑了吗？

下载最新 7z 压缩包，内含源码与运行环境，解压双击启动器即可使用。
